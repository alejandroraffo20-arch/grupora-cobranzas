⚙️ Configuración inicial

Para poner en marcha el sistema, necesitás configurar tu base de datos Supabase. Esto se hace una sola vez.

1. Crear cuenta en Supabase Ingresá a supabase.com → New Project → elegí un nombre y contraseña
2. Obtener credenciales En tu proyecto: Settings → API → copiá la "Project URL" y la "anon public" key
3. Crear las tablas Entrá a SQL Editor y ejecutá el script supabase_setup.sql que viene junto a este archivo
